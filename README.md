# Content Lab

Клиентское Next.js приложение для пошаговой сборки контент-системы: продукт, ЦА, рубрики, банк идей, контент-план и баланс.

## Запуск

```bash
npm install
npm run dev -- -p 3001
```

## Интеграции

Приложение статическое и хранит данные в `localStorage`, пока не подключены интеграции.

- OpenAI: добавьте API key и модель в блоке "Интеграции". Текстовые поля и кнопки автозаполнения используют Responses API.
- Google Sheets: для чтения публичной таблицы можно хранить Spreadsheet ID и API key.
- Запись в Sheets и создание событий Calendar из статического клиента лучше делать через Google Apps Script Web App. Шаблон лежит в `google-apps-script.example.js`.

В Apps Script нужно привязать скрипт к нужной таблице, вставить код из шаблона, развернуть Web App и добавить его URL в приложении.
